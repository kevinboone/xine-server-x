File scanner control

The current index file is %%index_file%%.

The current audio file root is %%root%%.

Start quick scan. Files that have been added or modified since the last scan will be scanned and added to the index. Files that are in the index, and have been removed from the filesystem, will be removed from the index. This operation will not even begin if there is not already an index in place.

Start full scan. The existing index will be replaced by a new one; all playable files will be scanned. This can be quite time-consuming. The existing index will remain available under the new one is complete.
